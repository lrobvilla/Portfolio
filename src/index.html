Linear algebra library prototype

Written in C++. This is a prototype for a linear algebra library, it utilizes operator overloading to facilitate operations between matrices and vectors. It was written as part of a little investigation project for a workshop centered around the QR matrix factorization algorithm. The main goal of the code was to compare two variants of the QR algorithm, one based on Householder reflectors and the classic version based on Gram-Schmidt. Current version is a little too rigid, needs polishing.
